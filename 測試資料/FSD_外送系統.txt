【線上外送訂餐系統 - 系統功能設計說明書 (FSD)】

1. 會員帳戶模組
- FUN-001：手機註冊與簡訊驗證 API。整合簡訊網關發送 6 位數驗證碼，並限制驗證碼時效為 5 分鐘。
- FUN-002：LINE 聯名登入功能。串接 LINE Login API 獲取 OpenID，並與本系統帳戶進行綁定。

2. 店家與搜尋模組
- FUN-003：LBS 定位與餐廳篩選服務。使用 PostGIS 計算經緯度距離，僅篩選距離店家 5 公里內之餐廳。
- FUN-004：Elasticsearch 模糊搜尋引擎。支援注音模糊比對、拼音糾錯，並建立餐點與菜系之搜尋索引。

3. 交易結帳模組
- FUN-005：購物車與備註處理邏輯。支援在資料庫暫存購物車內容，並記錄每個餐點品項之規格標籤。
- FUN-006：綠界/LINE Pay 支付金流整合。封裝金流 API 請求與 Webhook 回呼接收，處理交易成功與失敗之訂單狀態更新。

4. 物流配送與即時地圖模組
- FUN-007：即時訂單狀態 WebSocket 推送。當店家接單或外送員變更狀態時，透過 WebSocket 即時更新前端畫面。
- FUN-008：Google Maps API 地圖軌跡追蹤。於前端嵌入地圖元件，並以 10 秒為間隔輪詢外送員之 GPS 經緯度座標更新軌跡。
